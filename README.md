PTT MobilePay自動化爬取與主流電子支付聲量分析

開發自動化輿情監測工具，取代人工點閱，快速量化台灣主流電子支付品牌（街口、LINE Pay、一卡通）於社群論壇上的聲量佔比。

# 1.核心技術：Requests / BeautifulSoup / Pandas / Matplotlib
# 2.技術架構亮點：
**動態頁碼定位：自動解析網頁 ‹ 上頁 標籤以即時動態獲取最新的板塊頁數，免除硬編碼網址缺點。
**道德爬蟲設計：配置User-Agent瀏覽器模擬與 time.sleep(0.5) 控制請求頻率，避免觸發阻擋機制或造成伺服器過載。
**輿情數據視覺化：透過Pandas一鍵完成不區分大小寫的特徵篩選，並使用Matplotlib繪製雙子圖（長條圖與圓餅圖），並設定中文字型配置防碎碼。
# 3.市場洞察結果：成功自動化分析文本，量化指出LINE Pay聲量最高佔比40.9%奪冠，iPASS一卡通以34.8%緊追在後。
